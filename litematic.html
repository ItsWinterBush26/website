How to Install Litematica in Zalithlauncher, Zalithlauncher2 and others (Android)

What you‚Äôll need

  • Zalithlauncher2 or your prepared one (latest recommended build)
  • Minecraft Java Edition any version will work
  • Fabric loader (Litematica needs Fabric)
  • Litematica .jar and its dependency Malilib .jar
  • A file manager on Android (Files, ZArchiver, or MT Manager)

Step 1 ‚Äî Install Fabric Loader

  1. Open Zalithlauncher2.
  2. Click on Download icon ‚Üí Install Minecraft with fabric loader selected.
  3. Choose the Minecraft version that matches your Litematica build (e.g., 1.21.8) and tap Install.
  4. After installation, you should see an entry like fabric-loader-1.21.8 in your version list.

Step 2 ‚Äî Locate your game folder

In Zalithlauncher2, go to files Zalithlauncher2 icon (üìÅ) on the main screen to open the game directory. Navigate to:

If a mods folder doesn't exist, create it.

Step 3 ‚Äî Download and add the mods

  1. Download these .jar files from trusted sources (Modrinth or CurseForge):
    • Litematica (matching Minecraft version)
    • Malilib (matching version)
  2. Using your file manager, move the .jar files into: 
    /android/data/com.movtery.zalithlauncher.v2/android_data/files/.minecraft/versions/1.21.8 Fabric/mods/

Tip: Keep files named clearly, e.g., litematica-1.21.8.jar and

malilib-1.21.8.jar.

Step 4 ‚Äî Launch the game

  1. Open Zalithlauncher2 and select the Fabric entry you installed (e.g., fabric-loader-1.21.8).
  2. Tap Play.
  3. Once in-game, open the Mods menu to verify that Litematica and Malilib appear.

Step 5 ‚Äî Using Litematica

Controls and basic usage:

  • Open the Litematica menu with M in your controls (in-game).
  • To load schematics, place examplebuold.litematic files in: 
    /android/data/com.movtery.zalithlauncher.v2/android_data/files/.minecraft/versions/1.21.8 Fabric/schematics/
    (Create the schematics folder if needed.)
  • In-game ‚Üí Litematica ‚Üí Load Schematics ‚Üí select your file and adjust placement.
